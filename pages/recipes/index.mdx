# Recipes

Recipes are ready-to-follow blueprints for automating real business tasks with AI agents. Each recipe explains exactly what the agent does, what tools it needs, and what results to expect.

<Callout type="info">
These recipes are written for **business users** — no coding required. Each one can be implemented using the tools in the [Tools](/tools) section, most without writing a single line of code.
</Callout>

## How to read a recipe

Each recipe follows the same format:

1. **The problem** — what painful manual work this replaces
2. **The agent's job** — exactly what it does, step by step
3. **Tools needed** — what the agent requires to operate
4. **Expected results** — time savings and outcomes from real deployments
5. **Getting started** — the fastest way to implement this today
