## The three-layer mental model

Every AI agent system has three layers:

```
┌─────────────────────────────────┐
│         YOUR APPLICATION        │  ← What users see and interact with
├─────────────────────────────────┤
│         AGENT HARNESS           │  ← Orchestration, memory, tools, routing
├─────────────────────────────────┤
│         AI MODEL                │  ← The reasoning engine (Claude, GPT, etc.)
└─────────────────────────────────┘
```

You don't have to build all three yourself — that's what the tools in this cookbook are for. But understanding what lives at each layer makes you dramatically better at both using and building agent systems.
